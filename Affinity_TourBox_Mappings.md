# TourBox Preset for Affinity Studio

This document provides a complete mapping of TourBox buttons to Affinity Studio shortcuts. Since TourBox presets use a proprietary format, use this as a guide to manually configure your preset in the TourBox Console software.

## Button Mappings
- **A**: Move Tool (V)
- **B**: Node Tool (A)
- **C**: Pen Tool (P)
- **D**: Shapes Tool (U) - Rectangle/Ellipse/Rounded Rectangle cycle
- **E**: Text Tool (T) - Artistic/Frame Text cycle
- **F**: Zoom Tool (Z)
- **G**: Fill Tool (G) / Flood Fill
- **H**: Color Picker Tool (I)

## Toggle Mappings
- **Toggle 1**: Selection Brush Tool (W) / Flood Select
- **Toggle 2**: Paint Brush Tool (B)

## Dial Mappings
- **Dial 1 (Clockwise)**: Increase Brush Width (])
- **Dial 1 (Counter-Clockwise)**: Decrease Brush Width ([)
- **Dial 2 (Clockwise)**: Increase Opacity (9 - approx 90%)
- **Dial 2 (Counter-Clockwise)**: Decrease Opacity (8 - approx 80%)

## Wheel Mappings
- **Wheel 1 (Up)**: Zoom In (Ctrl + =)
- **Wheel 1 (Down)**: Zoom Out (Ctrl + -)
- **Wheel 2 (Up)**: Redo (Ctrl + Shift + Z)
- **Wheel 2 (Down)**: Undo (Ctrl + Z)

## Button Combination Mappings
- **A + E**: Crop Tool (C)
- **A + F**: Marquee Selection Tools cycle (M)
- **A + G**: Freehand Selection Tool (L)
- **B + E**: Point Transform Tool (F)
- **B + F**: Contour Tool (O)
- **B + G**: Corner Tool (C)
- **C + E**: Pencil Tool (N)
- **C + F**: Knife Tool (K)
- **D + E**: Shape Builder Tool (S)
- **D + F**: Picture Frame Tools cycle (K)
- **E + G**: Color Replacement Brush / Vector Flood Fill Tool (R)
- **F + H**: Healing Tools cycle (J)
